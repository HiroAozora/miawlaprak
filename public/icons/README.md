# 📁 Folder Icons untuk PWA

Taruh file icon kamu di sini dengan nama berikut:

| File | Ukuran | Dipakai untuk |
|---|---|---|
| `icon-72x72.png` | 72×72 | Android (legacy) |
| `icon-96x96.png` | 96×96 | Android |
| `icon-128x128.png` | 128×128 | Chrome Web Store |
| `icon-144x144.png` | 144×144 | Windows |
| `icon-152x152.png` | 152×152 | iOS Safari |
| `icon-192x192.png` | 192×192 | Android (utama) |
| `icon-384x384.png` | 384×384 | Android |
| `icon-512x512.png` | 512×512 | Android (splashscreen) |

## 💡 Tips

Kalau kamu sudah punya satu file logo (mis. 512×512), kamu bisa generate semua ukuran sekaligus di:
- https://www.pwabuilder.com/imageGenerator
- https://realfavicongenerator.net/
- https://favicon.io/

Upload logo kamu → download semua ukuran → taruh di folder ini.
